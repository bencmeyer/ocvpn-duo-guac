║          GITHUB + DOCKER HUB QUICK SETUP (Copy & Paste Commands)              ║
╚═══════════════════════════════════════════════════════════════════════════════╝

STEP 1: Create GitHub Repository
═════════════════════════════════════════════════════════════════════════════════
1. Go to: https://github.com/new
2. Repository name: openconnect-vpn
3. Description: OpenConnect VPN + Guacamole all-in-one container for Unraid
4. Visibility: Public
5. DON'T initialize README (we have one)
6. Click: Create repository
7. Copy the repository URL

STEP 2: Push to GitHub
═════════════════════════════════════════════════════════════════════════════════
cd /tmp/vpn-test-environment

# Replace with your actual repo URL from Step 1
git remote add origin https://github.com/YOUR_USERNAME/openconnect-vpn.git

git branch -M main

# This will prompt for your GitHub credentials
# Use: Personal Access Token (not password) - get from https://github.com/settings/tokens
git push -u origin main

✅ Your code is now on GitHub!

STEP 3: Create Docker Hub Repository
═════════════════════════════════════════════════════════════════════════════════
1. Go to: https://hub.docker.com/
2. Click: Create Repository
3. Name: openconnect-vpn-all
4. Description: All-in-one OpenConnect VPN + Guacamole gateway for Unraid
5. Visibility: Public
6. Click: Create

STEP 4: Push Docker Image to Docker Hub
═════════════════════════════════════════════════════════════════════════════════
# Login (this will prompt for username/password)
docker login

# Tag your image (replace YOUR_USERNAME with your Docker Hub username)
docker tag openconnect-vpn-all:latest YOUR_USERNAME/openconnect-vpn-all:latest

# Push (this takes 5-10 minutes)
docker push YOUR_USERNAME/openconnect-vpn-all:latest

# Verify it worked
docker pull YOUR_USERNAME/openconnect-vpn-all:latest

✅ Your image is now on Docker Hub!

STEP 5: Update Unraid Template
═════════════════════════════════════════════════════════════════════════════════
Edit vpn-connect.xml:

Find this line:
  <Repository>openconnect-vpn-all:latest</Repository>

Change to:
  <Repository>YOUR_USERNAME/openconnect-vpn-all:latest</Repository>

Example: If your Docker Hub username is "john-doe", it becomes:
  <Repository>john-doe/openconnect-vpn-all:latest</Repository>

STEP 6: Copy Template to Unraid
═════════════════════════════════════════════════════════════════════════════════
scp vpn-connect.xml root@YOUR_UNRAID_IP:/boot/config/plugins/dockerMan/templates-user/

Then:
- Refresh Unraid UI (F5)
- Go to Docker → Add Container
- Select "openconnect-vpn" template
- Fill VPN_USER and VPN_PASS
- Click Create
- Access http://YOUR_UNRAID_IP:8080

✅ You're done! 🎉

═════════════════════════════════════════════════════════════════════════════════
                            OPTIONAL: Auto-Build Setup
═════════════════════════════════════════════════════════════════════════════════

Enable GitHub Actions to auto-build on every push:

1. Go to: https://github.com/YOUR_USERNAME/openconnect-vpn
2. Settings → Secrets and variables → Actions
3. Click "New repository secret"
4. Name: DOCKER_HUB_USERNAME
   Value: your-docker-hub-username
5. Click "New repository secret"
6. Name: DOCKER_HUB_TOKEN
   Value: (create at https://hub.docker.com/settings/security)

Now every push to GitHub will auto-build and push to Docker Hub! 🚀
